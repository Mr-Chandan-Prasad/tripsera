🔗 Integration Test Flow:

Step 1: Set Up QR Code in Admin

1. Go to Admin Panel

2. Login with: admin / admin123

3. Click "QR Settings" tab

4. Upload a QR code image or paste a URL

5. Click "Save QR Code"

6. Verify success notification appears

Step 2: Test QR Code in Payment

1. Go to Bookings Page

2. Complete the booking process

3. On payment page, select "QR Code" payment method

4. Verify QR code image appears

5. Check browser console for debug logs

Step 3: Test Real-time Updates

1. Keep payment page open

2. Go back to admin panel and change QR code

3. Save the new QR code

4. Return to payment page and refresh

5. Verify new QR code appears

Step 4: Test Clear Functionality

1. Go to admin panel QR settings

2. Click "Clear" button

3. Verify QR code is removed

4. Go to payment page and select QR code method

5. Verify "QR Code not available" message appears
