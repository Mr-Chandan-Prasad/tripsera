📋 Firebase Setup Checklist

✅ Step 1: Firebase Project Created

Create a new Firebase project at Firebase Console

Required: Firebase project with Firestore, Auth, and Storage enabled

✅ Step 2: Firebase Configuration

Update your Firebase config in src/config/firebase.ts

const firebaseConfig = { apiKey: "your-api-key-here", authDomain: "your-project-id.firebaseapp.com", projectId: "your-project-id", storageBucket: "your-project-id.appspot.com", messagingSenderId: "your-sender-id", appId: "your-app-id", measurementId: "your-measurement-id" };

✅ Step 3: Security Rules

Deploy Firestore and Storage security rules

# Deploy rules firebase deploy --only firestore:rules firebase deploy --only storage:rules
